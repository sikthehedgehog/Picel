picel_create_bitmap

PicelBitmap *picel_create_bitmap(int width, int height, int use_palette)

Creates a new bitmap with the given properties. The last parameter is zero for true color bitmaps and non-zero for paletted bitmaps. Returns a pointer to the bitmap object or NULL if something went wrong.

Remember to call picel_destroy_bitmap when you're done using the bitmap to free the resources it's using.

picel_load_bitmap

PicelBitmap *picel_load_bitmap(const char *filename)

Load the contents of an image file. The format is determined by examining the file header. Returns a pointer to the bitmap object if the file was loaded successfully or NULL if there was an error.

Remember to call picel_destroy_bitmap when you're done using the bitmap to free the resources it's using.

picel_save_png

int picel_save_png(const char *filename, const PicelBitmap *bitmap)
