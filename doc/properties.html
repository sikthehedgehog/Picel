picel_is_bitmap_paletted

int picel_is_bitmap_paletted(const PicelBitmap *bitmap)

Checks if a bitmap is using a palette. If the bitmap is true color, it returns zero. If the bitmap is using a palette, it returns non-zero.

picel_convert_bitmap

int picel_convert_bitmap(PicelBitmap *bitmap, int width, int height, int use_palette)

Resizes the bitmap and changes its format if needed. If the bitmap is made smaller, pixels from the right and/or the bottom are removed. If the bitmap is made bigger, pixels are added at the right and/or the bottom.

To convert from true color to paletted, first set the bitmap's palette using picel_set_color then call this function. The bitmap will use that palette and colors will be adapted accordingly.
