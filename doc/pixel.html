picel_get_raw_pixel

uint32_t picel_get_raw_pixel(const PicelBitmap *bitmap, int x, int y)

Retrieves the color of a given pixel in a bitmap. If the bitmap is true color, the color is returned in RGBA 8.8.8.8 format. If the bitmap uses a palette, the color is returned as a palette index between 0 and 255.

picel_get_raw_data_ptr

uint8_t *picel_get_raw_data_ptr(PicelBitmap *bitmap)

Returns a pointer to the raw pixel data. Pixels are stored linearly, left to right, then top to bottom, no gaps between lines. It's up to the program to not access data outbounds.

In true color bitmaps, each pixel takes up four bytes (each byte representing red, green, blue and alpha values, respectively). In bitmaps using a palette, each pixel is a single byte (representing a palette index).
